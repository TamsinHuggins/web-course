# JavaScript Essentials (Demos)

Open these `.html` files directly in a browser and use DevTools Console.

## Demos

1. `01-variables-and-types.html` — `let/const`, primitive types, undeclared assignment
2. `02-operators-and-strings.html` — arithmetic, assignment, concatenation, template literals
3. `03-tests-and-truthiness.html` — `if/else`, `switch`, `==` vs `===`, logical operators, ternary
4. `04-loops.html` — `for`, `while`, `do...while`, `for...of`, `for...in`, `break/continue`
5. `05-functions.html` — function declaration, return values, default params
6. `06-arrow-functions-and-scope.html` — arrow functions, callbacks, `var` vs `let` scope, destructuring, spread
7. `07-try-catch.html` — exceptions, `try/catch/finally`
