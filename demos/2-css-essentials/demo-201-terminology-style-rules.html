CSS terminology

Style rules, selectors, declaration blocks, and declarations.

Demo 01

Style rule

A style rule is a selector plus a declaration block. 

h2 { color: orange; }

Declarations

Declarations live inside { } and end in ;. Each is a property/value pair. 

.note {
  border-left: 4px solid orange;
  padding-left: 12px;
}

Selectors

  • Element selector: h2
  • Class selector: .note
  • ID selector: #special

Tip

Right-click → Inspect to see which rules apply in DevTools.
