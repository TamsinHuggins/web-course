# Creating HTML Content (Demos)

Open these `.html` files directly in a browser.

## Demos

- `01-semantic-tags.html` — semantic page structure (`header`, `nav`, `main`, `article`, `section`, `aside`, `footer`)
- `02-data-attributes.html` — custom `data-*` attributes and reading them via `dataset`
- `03-misc-html5-tags.html` — a few extra HTML5 elements (`details/summary`, `progress`, `meter`, `time`)
- `04-html5-forms.html` — form layout using `fieldset`/`legend` + modern input types and attributes
