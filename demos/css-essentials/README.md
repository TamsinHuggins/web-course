# CSS Essentials (Demos)

Open these `.html` files directly in a browser.

## Demos

1. `01-terminology-style-rules.html` — style rules / selectors / declarations
2. `02-simple-selectors.html` — element, class, id, universal selector
3. `03-relational-selectors.html` — descendant, child, adjacent sibling, general sibling
4. `04-attribute-selectors.html` — `[attr]`, `[attr=value]`, `^=`, `$=`, `*=`
5. `05-chaining-and-grouping.html` — chaining selectors vs grouping selectors
6. `06-pseudo-classes.html` — link states, `:focus`, `:first-child`, `:nth-child`
7. `07-pseudo-elements.html` — `::first-line`, `::first-letter`, `::before`, `::after` (incl counters)
8. `08-vendor-prefixes.html` — what they look like and how to approach them
9. `09-backgrounds-borders-text.html` — outline, overflow, radius, shadow, opacity, text shadow
