# Lab: Getting Started with HTML, CSS and JavaScript

Audience: technical graduates.

> **Update:** Module 1 is **demos only**. Labs and solutions for “Getting Started” have been removed and will appear in later modules (e.g. JavaScript Essentials, Async + REST).

## Setup

Open the demo files under `demos/getting-started/` in a browser.

Tip: If you use VS Code, you can use a “Live Server” style extension, or just double-click the HTML file to open it.
